A Device Agnostic Approach to Complex Site Navigation

The Details

This is a Joomla port, by Seth Warburton, of the original Flexnav by Jason Weaver (to whom all credit should go). I have simply made some minor modifications to ensure that the script and css target existing elements of Joomla's native menu module, mod_menu. If you are not using Joomla, use the original version not this one!

What the author says about the original…

The mock navigation you see above is a mobile-first example of using media queries and javascript to make a decent site menu with drop downs. Special attention is paid to touch screens using click events. This is something I use to test different navigation techniques and may change as I iterate over different solutions to the problem. Basically I want a simple model to build upon when working on sites from scratch.

More details to come as I keep testing. Fork on GitHub.
